# Branding Notes

The app follows the DaisyPod brand direction:

- Deep black base canvas.
- Smoky amethyst stacked panels.
- Warm cream text.
- Soft yellow for primary actions.
- Coral for destructive/high-emphasis warnings.
- Crisp 1px borders rather than blurry shadows.
- Slight 4px–8px radii rather than pill-shaped controls.
- Noto Sans for UI/body/action type.
- DynaPuff for the main brand lockup only.
- Delius for short playful notes.

No font files are bundled. The app currently references Google Fonts from `index.html` and falls back to system fonts if offline.
